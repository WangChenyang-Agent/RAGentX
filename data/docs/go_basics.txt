# Go 语言基础

Go（又称 Golang）是 Google 开发的一种静态强类型、编译型、并发型，并具有垃圾回收功能的编程语言。

## Go 语言特点

1. **简洁性**：语法简单，只有 25 个关键字
2. **并发支持**：原生支持协程（goroutine）和通道（channel）
3. **高性能**：编译型语言，执行效率接近 C/C++
4. **垃圾回收**：自动内存管理，无需手动释放
5. **跨平台**：支持多种操作系统和架构

## 变量声明

Go 语言有多种变量声明方式：

```go
var name string = "hello"
var age = 25
city := "Beijing"  // 短变量声明
```

## 常量

```go
const pi = 3.14159
const (
    StatusOK = 200
    StatusNotFound = 404
)
```

## 数据类型

- 基本类型：int, float, string, bool
- 复合类型：array, slice, map, struct
- 指针类型：*T
- 函数类型：func
- 接口类型：interface
